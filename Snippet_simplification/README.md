## __Overview__

VSCodeで使用するユーザースニペットのbody部分を設定しやすいように変換します。

```text
aaa
bbb
ccc

↓

"aaa",
"bbb",
"ccc"
```

---

## __Usage__

```shell
python user_snippet_simplification.py
```
